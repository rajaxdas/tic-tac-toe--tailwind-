How To Play Tic-Tac-Toe

Objective

main goal of Tic-Tac-Toe is simple be the first player to get three of your marks (either 'X' or 'O') in a straight line. This line can be horizontal, vertical, or diagonal.

Gameplay

  • The game is played on a 3x3 grid.
  • Player 'X' always starts first.
  • Player take turns clicking on an empty square to place their mark('X' or 'O').
  • You cannot place ('X' or 'O') on the boxes that is already occupied.

    • Winning the Game

    You win if you successfully place three of your marks in a row, column, or diagonal. When a player wins, the winning line of squares will be highlighted in green, and a status message will declare the winner.

    Example winning lines:

  • The row : [X] [X] [X]
  • The middle column : [O] [O] [O]
  • diagonals : 
        [X] [ ] [ ]
    [ ] [X] [ ]
    [ ] [ ] [X]

    • Drawing the Game

    If all nine squares on the board are filled, and neither player has achieved three marks in a row, the game ends in a draw (also called a tie or stalemate). When this happens, all squares will be highlighted in red, and the status message will indicate "It's a Draw !".

    Restarting The Game

    After a game ends (either by a win or a draw) or even during a game if you want to start over, simply click the "Restart" button below the board. This will clear the grid and reset the game, allowing 'X' to start again.
