How to Play Tic-Tac-Toe

Objective

The goal of Tic-Tac-Toe is to be the first player to place three of your marks (either 'X' or 'O') in a row. The row can be horizontal, vertical, or diagonal.

Gameplay

The game is played on a 3Ã—3 grid.

Player 'X' always goes first.

Players take turns placing their mark in an empty square.

A square cannot be used more than once.

Winning the Game

You win if you get three of your marks in:

A horizontal row:
Example: 

[X] [X] [X]
[ ] [ ] [ ]
[ ] [ ] [ ]

A vertical column:
Example:

[O] [ ] [ ]
[O] [ ] [ ]
[O] [ ] [ ]

A diagonal line:

[X] [ ] [ ]
[ ] [X] [ ]
[ ] [ ] [X]

When a player wins, the winning squares are highlighted (e.g., in green), and a message shows who won.

Drawing the Game

If all nine squares are filled and no one has three in a row, the game ends in a draw. The board is highlighted (e.g., in red), and a message says: "It's a Draw!"

Restarting the Game

After a game ends (either by a win or a draw), or at any time, you can click the "Restart" button. This clears the board and starts a new game, with 'X' going first again.
