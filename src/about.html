Description

This website features a classic Tic-Tac-Toe game implemented using fundamental web technologies: HTML, CSS, and JavaScript. It allows two players to compete locally on the same browser.

This project was developed as part of a Minor Project for 2nd Year CSE students at Government Polytechnic, Nanakpur.

Objectives

The development of this project aimed to practice and demonstrate understanding of several key web development concepts:

  • Structuring web content using semantic HTML5.
  • Styling web pages effectively with CSS3, including layout techniques like Flexbox.
  • Implementing interactive logic using JavaScript (ES6+).
  • Manipulating the Document Object Model (DOM) to dynamically update the user interface.
  • Handling user events (like clicks).
  • Basic algorithm design for game logic (win/draw detection).
  • Organizing the project using multiple files (HTML, CSS, JS).

We hope you enjoy playing our version of Tic-Tac-Toe!
