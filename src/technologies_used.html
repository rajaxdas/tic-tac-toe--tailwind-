Technologies Used

This project utilizes fundamental web technologies to create an interactive Tic-Tac-Toe game:

  • HTML (HyperText Markup Language):

    Used to structure the content of the web pages, defining elements like headings, paragraphs, lists, buttons, and the containers for the game board.

  • CSS (Cascading Style Sheets):

    Used to control the visual presentation and layout. This includes setting colors, fonts, spacing, element sizes, and using Flexbox to arrange the game grid and center content. The specific styles for game states (winner, draw) are also handled by CSS.

  • Java Script

    Powers the game's logic and interactivity.

    It Handles :

  • Dynamically creating the game board elements.
  • Responding to user clicks (Event Handling).
  • Updating the game state (tracking moves).
  • Checking for win conditions or draws.
  • Resetting the game.

  • Google Fonts :

    Used to import and apply the 'Roboto' font for a consistent and modern look.

  • Browser Developer Tools:

    Essential tools (available in browsers like Chrome, Firefox, Edge) used for debugging JavaScript, inspecting HTML elements, and testing CSS styles during development.
